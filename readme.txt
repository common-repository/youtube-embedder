== Description ==

Youtube Embedder lets you embed a youtube video in a post by specyfing youtube address:

[youtube http://www.youtube.com/watch?v=AABBCCDDEEFF 300 200]

You can omit width and height if you want, so it will look like:

[youtube http://www.youtube.com/watch?v=AABBCCDDEEFF]

== Installation ==

1. Upload all files to the `/wp-content/plugins/embedder/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
